Bem-vindo ao curso de Programação 1! Neste site, você encontrará recursos para aprender os conceitos fundamentais de programação com Python, como estruturas de controle, tipos de dados e funções. O curso é composto por vídeo-aulas, exercícios práticos e um cronograma de atividades para guiar seu aprendizado. Cada aluno gravará uma vídeo-aula sobre um tópico específico. O objetivo é proporcionar uma experiência interativa e sólida para desenvolver suas habilidades em programação.

Introdução ao Python e Sintaxe Básica

Neste bloco, você aprenderá a instalar o Python e escrever seu primeiro programa "Hello, World!". Vamos explorar a sintaxe básica da linguagem, como variáveis, operadores e tipos de dados. Também abordaremos a entrada e saída de dados utilizando input() e print(). Este é o primeiro passo para começar sua jornada no mundo da programação.

Estruturas de Controle

Aqui, você vai aprender a usar estruturas de controle fundamentais, como condicionais (if, else, elif) e laços de repetição (for, while). Também veremos operadores de comparação e comandos de controle de fluxo como break, continue e pass. Essas ferramentas são essenciais para tornar seus programas interativos e dinâmicos.

Listas e Strings

Neste bloco, exploraremos dois dos tipos de dados mais importantes em Python: listas e strings. Você aprenderá a criar, modificar e acessar listas, além de manipular strings utilizando métodos como .lower(), .replace(), e o fatiamento. Essas habilidades são fundamentais para trabalhar com dados de maneira eficiente e flexível.

Funções

A definição e o uso de funções são o foco deste bloco. Você aprenderá a criar funções com def, passar parâmetros, e retornar valores. Além disso, abordaremos o conceito de escopo de variáveis, como locais e globais, para entender como organizar melhor o seu código e facilitar a reutilização de lógica.
